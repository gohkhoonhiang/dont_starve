 What is this site about?

When I first started playing DST, I found myself frequently checking the Fandom crockpot recipes to find out what recipes are available for a particular ingredient/material.

The Fandom page was great, but it is difficult to search and filter due to the ingredients/materials being rendered with icons.

I decided to create a simple website like this one to allow easier search and filter.

 Where do you get your data from?

As mentioned in the previous answer, I got my data from the Fandom crockpot recipes.

I downloaded the HTML source, normalized some values, and exported the data into CSV format. Then I cleaned up the CSV data manually, eg. to separate the recipes that allow for different alternative materials in different DLC versions. Finally, the cleaned up data are exported to JSON format, which is then served to the website.

Other sources of data:

The Starver's Guide to Self-Feeding Farms by QuartzBeam

As some of the data cleanup is automated, there may be error in interpreting the raw data. Feel free to submit a request for change here.

 Which browsers are supported?

This site has only been tested on MacOS Safari (Version 14.0.2 (16610.3.7.1.9)). No guarantee that it will work on other browsers.
